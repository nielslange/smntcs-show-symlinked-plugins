== Description ==

Prevent accidentally deleting or updating symlinked plugins.

== Installation ==

1. Upload `smntcs-show-symlinked-plugins` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
